# 🎨 Khmer Heritage App - Giao Diện Mới

## ✨ Tổng Quan

Ứng dụng di sản văn hóa Khmer Nam Bộ với giao diện hiện đại, đẹp mắt và đồng bộ.

---

## 🚀 Quick Start

```bash
npm install
npm start
```

Chọn platform: `a` (Android), `i` (iOS), `w` (Web)

---

## 📱 Màn Hình

| Màn hình | Màu chủ đạo | Highlights |
|----------|-------------|------------|
| 🏠 **Home** | Gold | Hero, Stats, Categories, Quick Actions |
| 🏛️ **Heritage** | Gold | Search, Filter, Heritage List |
| 🔍 **Explore** | Turquoise | Collections, Categories, Featured |
| 👤 **Profile** | Gold | Avatar, Stats, Settings, Languages |
| 🗺️ **Map** | Turquoise | Map Placeholder, Filter, Locations |

## 🎨 Design System

### Màu Sắc
- **Primary**: #D4AF37 (Gold)
- **Accent**: #4ECDC4 (Turquoise)
- **Success**: #10B981
- **Warning**: #F59E0B
- **Error**: #EF4444

### Components
- `Button` - 5 variants, 3 sizes
- `Card` - 3 variants
- `Badge` - 6 variants
- `SearchBar` - Focus state, clear button
- `HeritageCard` - Image, badges, stats
